# Foobar ![CI status](https://img.shields.io/badge/build-passing-brightgreen.svg)

Foobar is a Python library for doing awesome things with strings.

## Installation

### Requirements
* Linux
* Python 3.3 and up

`$ pip install foobar`

## Usage

```python
import foobar

foobar.awesome_method('baz')
```

## Development
```
$ virtualenv foobar
$ . foobar/bin/activate
$ pip install -e .
```
